\doxysubsection{Detailed Description}
A class that represents a 4D vector, derived from the \doxylink{classvec3}{vec3} class. 

The fourth dimension, w, is used to distinguish between points and directions in 3D space. When w is 1, the vector represents a point in 3D space. When w is 0, the vector represents a direction.
